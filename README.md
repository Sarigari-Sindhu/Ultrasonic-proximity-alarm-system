# Ultrasonic-proximity-alarm-system

##  Overview
This project measures distance using HC-SR04 ultrasonic sensor and provides multi-level proximity alerts.

##  Technologies Used
- Arduino Uno
- Embedded C
- Ultrasonic Sensor (HC-SR04)

## Features
- Real-time distance measurement
- LCD display output
- Multi-level buzzer alerts
